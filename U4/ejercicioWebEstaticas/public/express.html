¿Que es Express?

Express.js es un framework de Node.js podremos instalarlo como un paquete en el directorio de nuestro proyecto.
Es también de código abierto y con licencia MIT, su primera versión fue lanzada en 2010

Como Instalar Express

Podremos instalarlo como un paquete en el directorio de nuestro proyecto.
npm init ó npm init --yes.
Se nos generará un paquete llamado "package.json", nuestro proyecto Node.js está creado correctamente, para finalizar utilizaremos el comando npm install express --save para instalar el paquete EXPRESS.
Utilizamos "--save", para salvar esa dependencia en nuestro "package.json" de forma explicita.

Como crear nuestra primera app en Express

1º Creamos una const donde guardaremos las librerias de express.
2º Creamos otra constante donde guardaremos en la variable "app", las librerias de express.
3º Creamos en otra constante el puerto donde vamos a querer que se redireccione y podramos utilizarlo en la web.
4º Con la variable app, utilizaremos el metodo get, donde pasaremos por parametro la petición, y la respuesta,cuando entre a la web nos mostrara el mensaje que se visualiza en la imagen.
5º Volvemos a utilizar la variable app con el método get, cuando introduzacamos en la url la siguiente informacion "http://localhost:8080/'textoEjemplo'", como respuesta enviaremos la siguiente informacion en formato JSON.
6º Para finalizar pondremos el puerto del servidor, y una funcion flecha donde introduzcamos por consola un mensaje de que hemos entrado en ese puerto.

Como iniciar nuestro servidor node

Para poder iniciar nuestro servidor node, tendremos que:
1- Abrir nueva terminal
2- Entrar en el mismo repositorio donde esta el "app.js".
3- Insertar en la terminal "node .\'nombreArchivo'.js"
4- A disfrutar del framework Express!!.
