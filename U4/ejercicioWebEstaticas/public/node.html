¿QUE ES NODE?

Node es un “reproductor” que abre documentos escritos en el lenguaje JavaScript y “reproduce” (ejecuta) su contenido paso a paso. Existen módulos de Node (instalables) que extienden sus funcionalidades, como “electron-packager” que nos permite empaquetar programas para que puedan ser ejecutados de manera independiente, permitiéndonos crear nuestras propias aplicaciones de escritorio.

Como Instalar Node

La instalación de Node es sencilla utilizando el paquete de instalación disponible en el sitio web oficial de NodeJS: https://nodejs.org/en/ Descargar el instalador del sitio web Ejecutar el instalador, seguir los pasos del instalador. Reiniciar tu máquina Probar NodeJS desde el terminal, viendo su versión usando el siguiente comando: node -v

Como crear nuestro primer servidor Node

1º Importamos el módulo http, donde posteriormente crearemos un servidor con una función con dos parámetros:
- req: Es una patición.
- res: Es la respuesta.
2º Escribimos la cabecera con la petición con el código estado 200, ponemos el tipo de contenido que será, en este caso "text/plan".
3º Como respuesta, enviamos un "Hola Mundo"!
4º Escucharemos("server.listen") por el puerto, en este caso 8080.

Como iniciar nuestro servidor node

Para poder iniciar nuestro servidor node, tendremos que:
1- Abrir nueva terminal
2- Entrar en el mismo repositorio donde esta el "app.js".
3- Insertar en la terminal "node .\'nombreArchivo'.js"
4- A disfrutar del servidor NODE!!.
